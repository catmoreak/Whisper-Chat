# WhisperChat

A simple, anonymous chat application built with React,tailwind css  and Firebase. Create private chat rooms and communicate securely without leaving traces.

## Features

- **Anonymous Chat**: No sign-up required, just create or join a room
- **Secure Rooms**: 5-digit room codes for private conversations
- **Real-time Messaging**: Instant message delivery using Firebase
- **Online User Count**: See how many people are in your room
- **Modern UI**: Clean, dark blue theme with smooth animations using Tailwind CSS
- **Image upload**- Through the Img bb 
- **Encryption and Decrytion**- SHA256 encryption and decrytion for messages only (Images are not encryted)

## About

WhisperChat is designed for casual, anonymous conversations. It's perfect for quick chats without the hassle of accounts or personal information. Just create a room or join an existing one with a simple code.No need for any signup using email or Phone number.

## Security Note

Please avoid sharing sensitive information like phone numbers, addresses, or payment details in the chat. This app is for casual, anonymous conversations only.

## Technologies Used

- **Frontend**: React 18, TypeScript, Tailwind CSS
- **Backend**: Firebase Realtime Database
- **Icons**: Lucide React
- **Build Tool**: Vite
- **Styling**: Tailwind CSS with custom animations
